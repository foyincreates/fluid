# Consolidated Performance & Benchmark Reports

This document serves as the canonical index for all performance and benchmark
reports generated by the Fluid platform.  Individual report files live in this
`docs/reports/` directory rather than scattered across sub-project roots.

## Summary

### Migration Benchmark

- Execution time: **3524 ms**, memory overhead: **0.07 MB**

### Tokio Runtime Performance

- Best config: `high_concurrency` (20 worker threads, 80 max-blocking)
- Peak throughput: **18,110 RPS** at **0.06 ms** avg latency

## Generating New Reports

```bash
# Migration benchmark (from server/)
cd server && pnpm run benchmark:migration

# Tokio performance (from fluid-server/)
cd fluid-server && cargo bench
```

Move any generated `.md` reports into `docs/reports/` and update the table
above.
